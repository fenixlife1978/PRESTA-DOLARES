# Préstamos de Socios

WebApp moderna para gestión de préstamos, 100% responsive, lista para desplegar en Vercel.  
Inspirada en Starbucks y lista para ampliarse con base de datos y APIs.

**Cómo usar**  
1. Sube estos archivos a tu repositorio de GitHub.
2. Conéctalo con Vercel; detectará Next.js de inmediato y hará deploy.  
3. Si deseas soporte de base de datos más adelante, dímelo y te ayudo con `/prisma` y `/pages/api` para tus operaciones.

Desarrolla 100% ONLINE, sin instalar nada.
